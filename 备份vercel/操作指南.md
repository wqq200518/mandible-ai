# CBCT影像诊断系统 - 前后端集成操作指南

## 项目概述

本项目是一个完整的CBCT影像诊断系统，包含前端（Next.js + React）和后端（Node.js + Express + PostgreSQL）。

## 项目结构

```
网页项目/
├── frontend/              # 前端项目
│   ├── src/
│   │   ├── app/         # 页面组件
│   │   ├── lib/         # 工具库和API服务
│   │   └── components/  # UI组件
│   ├── package.json
│   └── next.config.ts
├── backend/             # 后端项目
│   ├── models/          # 数据库模型
│   ├── routes/          # API路由
│   ├── middleware/       # 中间件
│   ├── config.js        # 配置文件
│   ├── server.js        # 服务器入口
│   └── package.json
└── database.sql        # 数据库初始化脚本
```

## 系统要求

- Node.js 18.0+
- PostgreSQL 15.0+
- pnpm (包管理器)
- DBeaver (数据库管理工具)

## 详细操作步骤

### 第一步：数据库设置

1. **安装PostgreSQL数据库**
   - 下载并安装PostgreSQL 15.0或更高版本
   - 设置数据库密码（建议使用postgres）

2. **使用DBeaver连接数据库**
   - 打开DBeaver
   - 创建新连接：
     - 驱动：PostgreSQL
     - 主机：localhost
     - 端口：5432
     - 数据库：postgres
     - 用户名：postgres
     - 密码：postgres（或您设置的密码）

3. **执行数据库脚本**
   - 在DBeaver中打开SQL编辑器
   - 打开`database.sql`文件
   - 执行整个脚本，创建数据库和表结构
   - 确认所有表创建成功

### 第二步：后端设置

1. **安装后端依赖**
   ```bash
   cd backend
   pnpm install
   ```

2. **配置环境变量**
   - 编辑`backend/.env`文件
   - 根据您的数据库设置修改以下参数：
     ```
     DB_HOST=localhost
     DB_PORT=5432
     DB_USER=postgres
     DB_PASSWORD=postgres
     DB_NAME=cbct_diagnostic
     ```

3. **启动后端服务**
   ```bash
   # 开发环境
   pnpm dev

   # 生产环境
   pnpm start
   ```

4. **验证后端服务**
   - 打开浏览器访问：http://localhost:8000/health
   - 应该看到：`{"status":"ok"}`
   - 后端API地址：http://localhost:8000/api

### 第三步：前端设置

1. **安装前端依赖**
   ```bash
   cd frontend
   pnpm install
   ```

2. **启动前端开发服务器**
   ```bash
   pnpm dev
   ```

3. **访问前端应用**
   - 打开浏览器访问：http://localhost:3000
   - 应该看到登录页面

### 第六步：前端API集成

前端已经集成了API服务，位于`src/lib/api.ts`文件中。

**使用示例：**

```typescript
import { authApi, patientsApi, imagesApi, aiApi } from '../lib/api'

// 登录
const login = async () => {
  const response = await authApi.login('admin', 'admin123')
  if (response.data) {
    console.log('登录成功', response.data.user)
  }
}

// 获取患者列表
const getPatients = async () => {
  const response = await patientsApi.getPatients()
  if (response.data) {
    console.log('患者列表', response.data)
  }
}

// 上传影像
const uploadImage = async (file: File, patientId: string) => {
  const formData = new FormData()
  formData.append('image', file)
  formData.append('patient_id', patientId)

  const response = await imagesApi.uploadImage(formData)
  if (response.data) {
    console.log('上传成功', response.data)
  }
}

// AI分析
const analyzeImage = async (imageId: string) => {
  const response = await aiApi.analyze(imageId)
  if (response.data) {
    console.log('分析结果', response.data)
  }
}
```

## 常见问题

### 1. 数据库连接失败

**问题**：后端启动时提示数据库连接失败

**解决方案**：
- 检查PostgreSQL服务是否启动
- 检查数据库用户名和密码是否正确
- 检查数据库名称是否正确
- 检查防火墙设置

### 2. 前端无法连接后端

**问题**：前端登录时提示网络错误

**解决方案**：
- 检查后端服务是否启动（访问http://localhost:8000/health）
- 检查前端API_BASE_URL配置（在src/lib/api.ts中）
- 检查浏览器控制台的CORS错误

### 3. 文件上传失败

**问题**：上传影像或模型时失败

**解决方案**：
- 检查uploads目录权限
- 检查文件大小限制
- 检查文件格式是否支持

### 4. AI分析失败

**问题**：AI分析时返回错误

**解决方案**：
- 检查AI模型路径配置
- 检查AI模型文件是否存在
- 检查AI模型服务是否启动
